# Welcome to SOLAS
SOLAS is a tabletop role-playing game system built to run adventures in any setting, while being easy to pick up for beginners and seasoned veterans alike. It is focused primarily on theater-of-the-mind action and adventure, with four core abilities for tailoring characters.
